# CV Download Formats

This folder contains downloadable versions of Dr. Morada's curriculum vitae in multiple formats.

## Available Formats

- **PDF** - `anthony-morada-cv.pdf` (Coming soon)
- **Word** - `anthony-morada-cv.docx` (Coming soon)

## Generating CV Files

To generate PDF and Word versions from the markdown CV:
```bash
cd assets
python3 convert_cv.py
```

See `/assets/convert_cv.py` for conversion script.
